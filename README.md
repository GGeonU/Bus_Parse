# 버스도착정보 애플리케이션
- Use 버스도착정보조회 API, 다음 지도 API, SQlite, GPS 
- 특정 정류장의 도착정보, 정류장 검색, 즐겨찾기 제공
- GPS를 이용한 근처 버스정류장 검색, 정류장 위치 지도 제공
